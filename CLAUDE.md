## Commands

```bash
pip install -r requirements.txt   # Install dependencies
python scripts/login.py           # Authenticate with GitHub via device flow (saves token to public/config.json)
python src/mini_copilot.py        # Start the interactive CLI REPL
ruff check .                      # Lint code
ruff format .                     # Format code
```

Pre-commit hooks run automatically on commit: ruff lint + format.

## Architecture

This is a Python CLI (mini-copilot) that provides an interactive terminal REPL for chatting with GitHub Copilot.

**Authentication flow:**
1. `scripts/login.py` runs GitHub OAuth Device Flow, saves the GitHub token to `public/config.json`
2. On startup, `src/mini_copilot.py` reads the GitHub token from `public/config.json`, then exchanges it for a short-lived Copilot token via `https://api.github.com/copilot_internal/v2/token`
3. The Copilot token is refreshed every ~24 minutes during the session

**API endpoints:**
- `https://github.com/login/device/code` — Device authorization
- `https://github.com/login/oauth/access_token` — Token polling
- `https://api.github.com/copilot_internal/v2/token` — Copilot token exchange
- `https://api.githubcopilot.com/chat/completions` — Chat completions (GPT-4o model)

**Key files:**
- `src/mini_copilot.py` — Interactive REPL: loads token, maintains conversation history, calls Copilot API
- `scripts/login.py` — CLI login utility (Python)
- `public/config.json` — Generated by login script, gitignored; contains `{ github_token, created_at }`
- `requirements.txt` — Python dependencies (requests)
