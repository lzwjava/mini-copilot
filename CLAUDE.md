## Commands

```bash
npm install          # Install dependencies
npm run login        # Authenticate with GitHub via device flow (saves token to public/config.json)
npm run dev          # Start Vite dev server (typically http://localhost:5173)
npm run build        # TypeScript compile + Vite bundle
npm run lint         # Run ESLint
npm run preview      # Preview production build
```

Pre-commit hooks run automatically on commit: Prettier formatting, ESLint with `--fix`, and `tsc --noEmit`.

## Architecture

This is a React 19 + TypeScript SPA that provides a web chat interface to the GitHub Copilot API.

**Authentication flow:**
1. `scripts/login.js` runs GitHub OAuth Device Flow in Node (avoids browser CORS restrictions), saves the GitHub token to `public/config.json` with a timestamp
2. On app load, `src/services/auth.ts` reads the GitHub token from `public/config.json`, then exchanges it for a short-lived Copilot token via `https://api.github.com/copilot_internal/v2/token`
3. A refresh loop runs every ~24 minutes (with a 1-minute safety buffer before expiry) to keep the Copilot token valid
4. Session tokens are persisted in `localStorage`

**API endpoints:**
- `https://github.com/login/device/code` — Device authorization
- `https://github.com/login/oauth/access_token` — Token polling
- `https://api.github.com/copilot_internal/v2/token` — Copilot token exchange
- `https://api.githubcopilot.com/chat/completions` — Chat completions (GPT-4o model)

**Key files:**
- `src/App.tsx` — Main chat UI, handles messages, token validation, login/logout
- `src/services/auth.ts` — Token storage, Copilot token fetching, refresh loop, device flow
- `scripts/login.js` — CLI login utility (Node.js, not bundled)
- `public/config.json` — Generated by login script, gitignored; contains `{ githubToken, createdAt }`

**Styling:** CSS variables for light/dark theme in `src/index.css`; chat layout in `src/App.css`.
